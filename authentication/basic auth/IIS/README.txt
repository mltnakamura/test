##################################################
 基本認証のIISの設定
##################################################

①IIS マネージャを開き、管理するサイトに移動します。

②[機能ビュー] で、[認証] をダブルクリックします。

③[認証] ページで、[基本認証] を選択します。

④[操作] ウィンドウで [有効] をクリックして、既定の設定の基本認証を使用します。
　（注意：匿名認証は必ず[無効]にします。匿名認証が優先される為）

⑤必要に応じて、[操作] ウィンドウで [編集] をクリックして、既定のドメインおよび領域を入力します。 

⑥[基本認証設定の編集] ダイアログ ボックスの [既定のドメイン] ボックスに、既定のドメイン（corp.example.com等）を入力します。 
　サイトへのログオン時にドメインを指定しなかったユーザーについては、この既定のドメインに対して認証が行われます。 

⑦[領域] ボックスに、領域を入力するか、空白のままにします。
　領域名には、既定のドメインに使用したのと同じ値を使用できます。

⑧[OK] をクリックして [基本認証設定の編集] ダイアログ ボックスを閉じます。
